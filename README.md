# 🕌 Masjid Al Ihsan Website

A professional profile and services website for Masjid Al Ihsan, featuring a **Modern, Soft Minimalist, & Elegant** design. This website is meticulously crafted to provide a comfortable, informative, and inclusive digital experience for all congregation members, ranging from youth to the elderly.

## 🚀 Key Features

- **Soft Minimalist Aesthetics:** A modern User Interface (UI) utilizing soft color palettes, glassmorphism effects, and soft shadows to deliver a clean and elegant look.
- **Fully Responsive:** Seamlessly adapts to any screen size, providing an optimal viewing experience across desktops, tablets, and smartphones.
- **Dynamic Prayer Times:** Real-time prayer schedule integration that automatically highlights the currently active prayer time according to Islamic jurisprudence (including the Dhuha/Syuruq phase).
- **Transparent Information:** Comprehensive details regarding the Mosque's Profile, Programs & Studies, Construction Progress Reports, and ZISWAF (Zakat, Infaq, Sadaqah, Waqf) services.
- **Direct Contact Integration:** Call-to-Action (CTA) buttons linking directly to the WhatsApp Center for donation confirmations and general inquiries.

## 🛠️ Tech Stack

- **HTML5** for semantic page structuring.
- **Tailwind CSS (via CDN)** for rapid, modern, and highly responsive utility-first styling.
- **Vanilla JavaScript** for interactive functionalities (mobile menus, scroll effects, dynamic tabs, and prayer times API integration).
- **Font Awesome** for crisp vector icons.
- **Google Fonts (Inter & Playfair Display)** for modern, legible, and professional typography.

## 📂 Project Structure

1. `index.html` - Main landing page (Hero section, Dynamic Prayer Times, Program Highlights)
2. `pages/profil.html` - History, Vision, Mission, and Organizational Structure
3. `pages/program.html` - Comprehensive list of mosque services and activities
4. `pages/pembangunan.html` - Construction progress, financial transparency, and documentation
5. `pages/infaq.html` - ZISWAF donation services featuring a 1-click account copy utility
6. `pages/kontak.html` - Interactive location map, WhatsApp Center, and email contact

## 💻 Getting Started (Installation)

Since this project runs entirely on the client side, no server-side installation or complex build steps are required.

1. Clone this repository:
   ```bash
   git clone https://github.com/Ihzaanandar/masjid-al-ihsan.git
   ```
2. Navigate into the project directory.
3. Open `index.html` using your preferred web browser (Google Chrome, Firefox, Edge, etc.).
   
   *Note: For the best development experience, it is highly recommended to use an extension like **Live Server** in VS Code.*
